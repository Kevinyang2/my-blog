# 博客项目

这是一个简单的静态博客网站，包含以下页面：

- 首页 (index.html)
- 关于页面 (about.html) 
- 博客页面 (blog.html)
- 联系页面 (contact.html)

## 功能

- 响应式布局
- 统一的CSS样式 (css/style.css)
- 简洁的页面导航

## 使用方法

1. 克隆本仓库
2. 打开index.html即可浏览网站
3. 你可以在https://kevinyang2.github.io/my-blog 访问到这个静态网页
